Overview

"My Grammar Plugin" is designed to help users who have visual impairment quickly identify and understand various grammar terms within their text by using intuitive keyboard shortcuts. This plugin aims to streamline the editing process, making it easier to pinpoint and address grammatical issues.

By default, the commands (gestures) for "My Grammar Plugin" are listed below. If these commands conflict with your system or preferred text editor, you can reassign these gestures within the plugin's settings or your text editor's keybinding configuration.
