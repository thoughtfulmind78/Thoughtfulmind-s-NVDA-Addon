Overview

"My Grammar Plugin" is designed to help users who have visual impairment quickly identify and understand various grammar terms within their text by using intuitive keyboard shortcuts. This plugin aims to streamline the editing process, making it easier to pinpoint and address grammatical issues.

By default, the commands (gestures) for "My Grammar Plugin" are listed below. If these commands conflict with your system or preferred text editor, you can reassign these gestures within the plugin's settings or your text editor's keybinding configuration.

---

Keyboard Shortcuts for Grammar Terms

Below are the default keyboard shortcuts to report or highlight specific grammar terms:

  • **Ctrl+Alt+S**: Reports/Highlights **Subject** of the sentence.
  • **Ctrl+Alt+V**: Reports/Highlights **Verb** in the current clause.
  • **Ctrl+Alt+O**: Reports/Highlights **Object** (direct or indirect).
  • **Ctrl+Alt+A**: Reports/Highlights **Adjective** phrases.
  • **Ctrl+Alt+D**: Reports/Highlights **Adverb** phrases.
  • **Ctrl+Alt+N**: Reports/Highlights **Noun** phrases.
  • **Ctrl+Alt+P**: Reports/Highlights **Prepositional** phrases.
  • **Ctrl+Alt+C**: Reports/Highlights **Conjunctions**.
  • **Ctrl+Alt+T**: Reports/Highlights **Tense** of the main verb.
  • **Ctrl+Alt+M**: Reports/Highlights **Modal verbs**.
  • **Ctrl+Alt+X**: Reports/Highlights **Common grammar errors** (e.g., subject-verb agreement, pronoun antecedents).
